ข้อดีข้อเสียของการทำ css utility

- runtime
  - pros
    - สามารถทำ dynamic styling ได้หลากหลาย
    - flexible ที่สุด
    - no setup

  - cons
    - ตอนโหลดครั้งแรกอาจจะนานที่สุด เพราะ js generate css อยู่

-buildtime
  - pros
    - performance ดีกว่า

  - cons
    - setup เยอะที่สุด
    - ทำ dynamic ได้ยากกว่า
    - ไม่ค่อย flexible
